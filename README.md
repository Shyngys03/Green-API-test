# WhatsApp API Tests

# Клонирование репозитория
```
git clone https://github.com/Shyngys03/Green-API-test.git
```
## Установка
```
npm install
```
## Настройка
Создать .env файл:

```
BASE_URL=
INSTANCE_ID=
API_TOKEN=
CHAT_ID=
```

## Запуск
```
npm test
```
или
```
npx jest
```
## Покрытие
- sendMessage
- getChatHistory
- getStateInstance

## Проверки
- статус коды
- обязательные поля
- авторизация инстанса
